# Vision

AI Project Kit exists to make AI-assisted software development structured, repeatable, and enjoyable.

Rather than starting every project from scratch or repeatedly explaining development preferences to AI coding assistants, AI Project Kit provides a reusable foundation that enables humans and AI to collaborate effectively.

The long-term vision is to create a development methodology where every new project begins with a shared understanding of the project's purpose, workflow, engineering standards, and collaboration model.

---

# Mission

Build a reusable project starter kit that enables humans and AI teammates to work together as an effective engineering team.

Every completed project should strengthen the kit, making future projects easier to start, easier to understand, and easier to maintain.

---

# Core Principles

## Human-led

Humans own the vision, priorities and final decisions.

AI exists to accelerate execution, not replace judgement.

---

## AI-first

Design repositories so AI coding assistants can quickly understand:

- the project
- the architecture
- the coding standards
- the next task

---

## Small Iterations

Break work into small, reviewable tasks.

Each task should produce a meaningful improvement while keeping the project in a working state.

---

## Living Documentation

Documentation is part of the product.

Project documents should evolve alongside the codebase and remain accurate throughout the project's lifetime.

---

## Continuous Improvement

Every project built using AI Project Kit should contribute improvements back into the kit.

The template should evolve continuously based on real-world experience.

---

# Success Criteria

AI Project Kit is successful when:

- New projects can be started in minutes.
- AI teammates understand the project with minimal prompting.
- Development feels organised and enjoyable.
- Projects remain maintainable months or years later.
- Lessons learned are captured and reused.

---

# Long-Term Vision

AI Project Kit should become a reusable foundation for building software with AI teammates, regardless of language, framework or technology.

The goal is not simply to build software faster.

The goal is to build better software through better collaboration between humans and AI.

---
