# Utility Scripts

This directory contains utility scripts for development and operations.

## Purpose

Scripts should be used for:
- Local development setup
- Database migrations
- Data import/export
- Deployment utilities
- Maintenance tasks

## Conventions

- Scripts should be self-documenting
- Include usage instructions in comments
- Should work on Unix/Linux/macOS systems
- Include error handling
- Log important actions

## Examples

- `setup.sh` - Initial environment setup
- `migrate.js` - Run database migrations
- `deploy.sh` - Deployment script
